在绝望中寻找希望
俞敏洪 / 中信出版社 / 2014-5-1 / 36.00元
2020-02-15 读过 / /
《在绝望中寻找希望》——俞敏洪亲笔书写50年心路历程，20载创业风云。 俞敏洪每天晚上12点之后会坚持写一篇文章，《在绝望中寻找希望》就是这些记录文字的呈现。全书精选了104篇文章，针对当下年轻人面临的一系列困扰，按青春、理想、心态、亲情、友情、人生、教育等主题,呈现出他对世界的看法和态度。在这个充斥着不安感的浮躁社会，给年轻人内心坚强的力量，让每个人思考应该怎样更积极地度过人生。 俞敏洪的文字有种朴实而坚定的力量，阅读他的文字，你能感受到向上的张力，昂扬的态度和智慧的人生哲学。

汤木 / 2014-6 / 32.00元
2020-02-15 读过 / /
◎《将来的你，一定会感谢现在拼命的自己》，是冯仑、张德芬、毕淑敏、柴静等人希望你早日能够看到的一本正能量书！ ◎ 100多篇真诚之作！十万多字的肺腑之言，经验之谈！激励千万读者、点赞无数的心灵治愈能量书，让内心浮躁、内心迷茫的人坚定内心、看清方向，去方向更辽阔的天空，为将来的自己赢得一片天地。 ◎《生命的奖赏从来不在起点》《没有计划的人一定被计划掉》《幸运的人总是幸运，倒霉的人总是倒霉》《别上了固执的当》《别让“忍不住”害了你》《善于等待的人，一切都会及时到来》《多一份磨砺，多一份强大》《破晓总是为了等待它的人来临》篇篇都很精彩，给人以启迪，你会感觉自己的心灵不断被敲击，被惊醒！你终会相信努力的意义，因为将来的那个你，一定会感谢现在拼命努力的自己！
